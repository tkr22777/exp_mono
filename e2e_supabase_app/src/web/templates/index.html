API Documentation
Available Endpoints:
Authentication:
  • POST /auth/register - Register a new user
  • POST /auth/login - Login a user
  • POST /auth/logout - Logout the current user
  • GET /auth/profile - Get the current user's profile
User Profiles:
  • GET /api/profiles/me - Get your profile
  • PATCH /api/profiles/me - Update your profile
  • PATCH /api/profiles/me/metadata - Update your profile metadata
  • GET /api/profiles/:id - Get a user's profile
System:
  • GET /api/health - Health check endpoint
